GScript

Игровой язык программирования

Как начать

  1. Скачай GScript
  2. Создай файл main.gs
  3. Запусти его

Пример GScript

player.hp = 40 when player.hp < 50 { say "LOW HP!" spawn Enemy } damage player 20 heal player move player forward

Документация

Переменные

set hp 100

Объекты

player.hp = 100 enemy.hp = 50

Условия

when player.hp < 50 { say "LOW HP!" }

Команды

spawn Enemy move player forward heal player damage player 20 say "Hello"

Циклы

loop 3 { spawn Enemy }
