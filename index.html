Как начать с GScript

  1. Скачайте GScript по ссылке ниже
  2. Создайте файл с именем main.gs
  3. Напишите первый код и запустите его

Примеры кода

spawn(Enemy)
player.hp = 100
if (player.hp < 50) {
    heal(player)
}

Документация

Переменные

Переменные в GScript хранят данные, такие как числа или текст. Пример:

hp = 100

Команды

  • spawn(Enemy): Создаёт врага в игровом мире
  • move(player, direction): Перемещает игрока в заданном направлении
